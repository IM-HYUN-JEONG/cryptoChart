# 프로젝트 소개
외부 오픈API와 통신하여 실시간 코인들의 잔액을 확인하는 서비스입니다.
<hr />

# 위 서비스에서 제공하는 핵심 기능은 1가지입니다.
- 대표 코인들의 실시간 잔액 및 차트 확인
<hr />

# 사용 기술스택 구현
- Recoil : atom을 사용하여 라이트/다크모드를 상태관리
- ApexCharts : 데이터를 시각화 해주는 차트
- API : API와 통신하여 실시간 데이터 fetch
